Exercise Tracker

User's exercise log:
<above URL>/api/exercise/user/{userId}/date1/<from>/date2/<to>/limit/<number>

{ } = required, < > = optional

from, to = dates (yyyy-mm-dd); limit = number

Example: <above URL>/api/exercise/user/5be3f498ed070b2ddca768b0/date1/2017-12-23/date2/2018-11-09/limit/2

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add
